## APP 说明

> Mysql 操作客户端

### 通用参数

|  参数   | 类型  |  必填   |  备注  |
|  ----  | ----  |  ----  |  ----  |
| **host**  | text | `是` | Mysql 主机 |
| **port**  | number | `是` | Mysql 端口 |
| **user**  | text | `是` | 账号 |
| **passwd**  | text | `是` | 密码 |
| **db**  | text | `是` | 数据库 DB |
| **sql**  | text | `是` | 执行的 SQL  |

## 动作列表

返回查询的数据
