## APP说明

> SSH Linux 服务器执行远程命令

## 动作列表

### 执行命令

**参数：**

|  参数   | 类型  |  必填   |  备注  |
|  ----  | ----  |  ----  |  ----  |
| **host**  | text | `是` | 主机地址 |
| **port**  | number | `是` | SSH 端口 |
| **user**  | text | `是` | 登录用户 |
| **passwd**  | text | `是` | 登录密码 |
| **shell**  | text | `是` | 执行命令 |

**返回值：**

```
执行命令的返回结果
```
